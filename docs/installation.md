# Installation

## Stable release

To install spectrapepper, run this command in your terminal:

```
pip install spectrapepper
```

This is the preferred method to install spectrapepper, as it will always install the most recent stable release.

If you don't have [pip](https://pip.pypa.io) installed, this [Python installation guide](http://docs.python-guide.org/en/latest/starting/installation/) can guide you through the process.

## From sources

The sources for spectrapepper can be downloaded from the Github repo.

You can clone the public repository:

```
git clone git://github.com/enricgrau/spectrapepper
```
