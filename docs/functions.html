spep.baseline_als(y, lam=100, p=0.001, niter=10)[source]¶

Calculation of the baseline using Asymmetric Least Squares Smoothing. This script only makes the calculation but it does not remove it. Original idea of this algorythm by P. Eilers and H. Boelens (2005), and available details at: https://stackoverflow.com/a/29185844/2898619

Parameters

  • y (list[float]) – Specra to calculate the baseline from.

  • lam (int) – Lambda, smoothness. The default is 100.

  • p (float) – Asymetry. The default is 0.001.

  • niter (int) – Niter. The default is 10.

Returns

Returns the calculated baseline.

Return type
